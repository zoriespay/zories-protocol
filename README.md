# Zories

> Recurring payments and payroll on Solana. Approve once, let it run forever.

Zories is a non-custodial payment infrastructure built on Solana that enables recurring USDC subscriptions, automated payroll, and scheduled payouts. Businesses, DAOs, startups, and developers can automate payments while users retain full control of their assets.

---

## 🚀 Overview

Traditional crypto payments require manual approvals for every transaction. Zories removes that friction by allowing users to authorize recurring payments once and automate the rest.

Designed for:

- 💼 Teams and payroll operations
- 🏪 Merchants and subscriptions
- 🌐 DAOs and communities
- 🤖 AI agents and automation
- 👨‍💻 Developers building on Solana
